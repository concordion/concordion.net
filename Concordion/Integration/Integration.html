Running Active Specifications

Concordion.NET integrates with one of the most popular testing frameworks, NUnit. Concordion.NET provides an NUnit-Addin that allows executing active specifications. Please, see the information below for more details on test execution:

  • To find out how to execute your automated tests, please visit the page on running Concordion.NET tests with NUnit.
  • Initialization code can be executed before test runs with the help of TestFixtureSetUp attributes in Concordion.NET fixture classes.
