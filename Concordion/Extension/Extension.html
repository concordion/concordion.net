Extension

A Concordion.NET extension introduces additional functionality to Concordion.NET. Extensions are able to:

  • Add custom commands to Concordion.NET
  • Act as listeners to Concordion.NET commands and processing events
  • Create resources in the Concordion.NET output folder
  • Add CSS or JavaScript to the Concordion.NET output
  • Read and write to files using different file suffixes
