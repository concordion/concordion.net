Displaying Results

After processing a document, Concordion outputs HTML based on the input document with the results marked up.

Commands

The assertEquals command reports successes and failures.

The assertTrue command reports successes and failures.

The verifyRows command reports missing and surplus rows.

Other

  • Exceptions can be thrown by any command. They are displayed in the document along with a toggleable stack trace.
  • Breadcrumbs are automatically added to documents based on the package hierarchy.
  • Styles are embedded in the document so that successes and failures are displayed correctly.
  • Images and stylesheets referenced by the document are copied to the output.
